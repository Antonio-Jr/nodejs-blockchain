# nodejs-blockchain

### Setting up the DHT

```
npm i -g grenache-grape
```

```
# boot two grape servers

grape --dp 20001 --aph 30001 --bn '127.0.0.1:20002'
grape --dp 20002 --aph 40001 --bn '127.0.0.1:20001'
```

``` 
# To execute please type the commands:
    npm install
    npm run server
    npm run client
```
